    Detailed process

    2.2.1 模型定义

    2.2.1.1 输入编码

    给定输入向量\(x∈R^D\),编码为一次脉冲的时间：

    \(t_i​=\text{encode}(x_i​)\)

    • \(x_i\)：原始输入的数值

    • encode：将数值转化成时间的映射函数

    • \(t_i\)：第 \(i\) 个神将元的发放spike的时间

      • 也就是输入神经元的在时间\(t_i\)发出唯一一次的\(spike\)

    2.2.1.2 网络结构

    • 输入层：\(D\) 个神经元（各发一次）

    • 输出层：二维网格 \(M×N\)，每个节点是一个 CD（coincidence detector）

      • CD：同时性检测的神经元→即当多个输入spike在时间上“接近同时到达”时才会发放的神经元

      • 在传统的ANN中，相似就是距离小。在这里相似就是spike时间可以对齐

    • \(CD_{ij}\)是一个输出神经元：

      • 具有：

        • 自己的delay参数：\(d^{(ij)}=\{d_{1,ij},d_{2,ij},...,d_{D,ij}\}\)

        • 自己的膜电位：\(u_{ij}(t)\)

        • 自己的发放时间：\(t_{ij}^{fire}\)

      • 对于每个\(CD_{ij}\)，它接受所有输入神经元的spike，每个输入都有一个delay

      • 每个CD都在做：能不能把这些spike对齐

    • 连接：每个输入 \(i\) 到每个输出 \(j\) 有一条突触，权重固定 \(w_{ij}\)​，delay 可学习 \(d_{ij}\)​

    2.2.1.3 神经元动力学

    输入神经元 \(j\) 的膜电位：

    \(u_j​(t)=∑_{i=1}^D​w_{ij}​ε(t−(t_i​+d_{ij}​))\)

    • \(ε(⋅)\)：PSP 核（如 alpha kernel）

    当 \(u_j​(t)≥θ\) 时，神经元在时刻 \(t_j^{fire}\)​ 发放（只看第一次发放）

    2.2.1.4 输入+输出

    输入：一组spike的时间

    输出：一个spike（最早的那个）

    实际使用：这个spike属于哪个神经元

    2.2.2 一次前向计算（决定winner）

    我们的目标是找到最先发放的神经元（winner）

    步骤：

    1. 所有输入在各自时间\(t_i\)发spike

    2. 通过delay的传播，产生PSP

    3. 每个输出神经元积分得到 \(u_j(t)\)

    4. 找到：$ j^*=\text{arg }\text{min​}_j\text{ }t_j^{fire}​$

    最早方法 = 最匹配（时间最对齐）

    2.2.3 学习规则

    （1）更新对象：delay

    \(d_{ij} \leftarrow d_{ij} + Δd_{ij}\)

    （2）Hebbian + 时间对齐思想

    目标：让到达时间对齐：

    \(t_i+d_{ij}\approx t_{j^*}^{fine}\)

    因此一个自然的更新是：

    \(Δd_{ij}​∝(t_{j^*}^{fine}−(t_i​+d_{ij}​))\)

    • 如果该输入到得太早 → 增大 delay（推迟）

    • 如果太晚 → 减小 delay（提前）

    （3）邻域更新（SOM核心）

    我们不仅更新winnier，也更新邻居：

    \(Δd_{ij}​=η⋅h_{j^*,j}​⋅(t_{j^*}^{fire}​−(t_i​+d_{ij}​))\)

    其中：

    \(h_{j^*,j}=\text{exp}(-\frac{||r_j-r_{j^*}||^2}{2\sigma^2})\)

    • \(r_j\)​：神经元在网格中的位置

    • \(σ\)：邻域半径（逐渐减小）

    （4）只用first spike

    所有计算只基于：

    \(t_j^{fire}\)

    忽略后续的spike

    2.2.4 完整的训练流程

    初始化：

    • \(d_{ij}​∼U(0,d_{max}​)\)

    • \(w_{ij}\)​ 固定（常数或小随机）

    • 阈值 \(θ\) 初期较小

    • 学习率 \(η\)、邻域 \(σ\)

    step1：编码

    \(x\rightarrow \{t_i\}\)

    step2：前向传播

    计算每个\(u_j(t)\)，得到发放时刻\(t_j^{fire}\)

    step3：选择winner

    $ j^*=\text{arg }\text{min​}_j\text{ }t_j^{fire}​$

    step4：更新delay（winner+邻域）

    \(Δd_{ij}​=η⋅h_{j^*,j}​⋅(t_{j^*}^{fire}​−(t_i​+d_{ij}​))\)

    其中：

    \(h_{j^*,j}=\text{exp}(-\frac{||r_j-r_{j^*}||^2}{2\sigma^2})\)

    step5：退火

    • \(η↓\)

    • \(σ↓\)

    • \(θ↑\)

    step6：停止条件

    • delay变化很小

    • 或者训练论述达到上限
